# 🕊️ SoulVault AI

> **Preserve your digital life. Deliver your memories. Live on through AI.**

---

## 🌍 Overview

SoulVault AI is a **digital legacy platform** that helps people securely store their digital assets, preserve personal memories, and create an AI-powered version of themselves that loved ones can interact with in the future.

In today’s world, our lives exist online — but when we’re gone, everything we leave behind becomes inaccessible, scattered, or lost.

**SoulVault AI ensures your digital life doesn’t disappear.**

---

## 🚨 Problem

* Digital assets (emails, wallets, accounts) are locked after death
* Families struggle to recover important information
* Personal memories and emotional connections are lost
* There is no human-centered solution for digital inheritance

---

## 💡 Solution

SoulVault AI combines **security + AI + emotional preservation** into one platform:

* 🔐 Securely store digital assets
* 👥 Assign trusted beneficiaries
* ❤️ Preserve memories (text, voice, video)
* 🧠 Train an AI that thinks and speaks like you
* ⏳ Automatically transfer access after inactivity

---

## ✨ Key Features

### 🔐 Digital Vault

* Store accounts, credentials, and recovery instructions
* End-to-end encryption for sensitive data
* Assign assets to individuals or all beneficiaries
* Secure access only after protocol activation

---

### 👥 Beneficiary System

* Add trusted individuals
* Define access levels (Full / Limited)
* Support for phone + email-based identity
* Controlled and secure data sharing

---

### ❤️ Memory Preservation

* Save messages as:

  * Text
  * Voice (planned)
  * Video (planned)
* Assign memories to specific people
* Emotional delivery after inactivity

---

### 🧠 AI Legacy Training

* Train AI with:

  * Personal stories
  * Values and beliefs
  * Communication style
* AI learns to respond like the real user

---

### 💬 AI Legacy Interaction

* Beneficiaries can **chat with an AI version of the user**
* Context-aware responses using memories + training
* Human-like tone (not robotic responses)
* Conversation history support

---

### ⏳ Inactivity Protocol

* Detects user inactivity
* Triggers verification process
* Grants access to beneficiaries securely
* Fully simulated in MVP

---

### 🎥 Future Features (Vision)

* AI Voice Cloning
* AI Video Calling (Digital Avatar)
* Scheduled Memory Delivery (future events)
* Blockchain-secured inheritance
* Legal digital will integration

## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/your-username/soulvault-ai.git
cd soulvault-ai
```

### 2. Install dependencies

```bash
npm install
```

### 3. Setup environment variables

Create `.env.local`:

```env
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
GEMINI_API_KEY=your_api_key
```

### 4. Setup database

* Run `upgrade_db.sql` in Supabase SQL editor

### 5. Run the app

```bash
npm run dev
```
